About the Project

The Hydrological Calculator is a web-based tool designed to assist engineers, researchers, and environmentalists in performing hydrological calculations. It features real-time rainfall intensity analysis, geographical mapping, and water flow estimations.

Technologies Used

  • Backend: Python (Django)
  • Frontend: HTML, CSS, JavaScript
